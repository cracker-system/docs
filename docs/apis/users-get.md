# ユーザー取得
ユーザー情報の取得を行います。

## 基本仕様
- エンドポイント : /users/{user_id}
- メソッド : GET

なおエンドポイントの{user_id}は省略可能です。省略した場合ユーザー一覧が取得できます。

## リクエストヘッダー
APIキーなどの共通の設定は「Cracker API概要」を参照してください。

## パスパラメータ

|パラメータ名|タイプ|必須|説明|
|----|----|----|----|
|user_id|文字列||特定のユーザー情報を取得する場合、ユーザーIDを指定します。|

## クエリパラメータ
なし

## リクエストボディ
なし

## レスポンスボディ

|パラメータ名|タイプ|説明|
|----|----|----|
|user_id|文字列|ユーザーID|
|user_name|文字列|ユーザー名|
|exhibit_id|文字列|ユーザーの所属展示|

レスポンスボディサンプル
```
{
    "user_id": "31ennichi",
    "user_name": "3-1 縁日",
    "exhibit_id": "31"
}
```
