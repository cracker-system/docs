# 展示削除
既存展示の削除を行います。

## 基本仕様
- エンドポイント : /exhibits/{exhibit_id}
- メソッド : DELETE

## リクエストヘッダー
APIキーなどの共通の設定は「Cracker API概要」を参照してください。

## パスパラメータ

|パラメータ名|タイプ|必須|説明|
|----|----|----|----|
|exhit_id|文字列|必須|削除する展示IDを指定します。|

## クエリパラメータ
なし

## リクエストボディ
なし

## レスポンスボディ
なし
